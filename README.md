Sushitter
=========

Sushitter by Ruby on Rails 

#Sushitterとは
ツイートされた文章を分析して、その文章がポジティブかネガティブかを判断し、その感情にあった寿司ゆきをアイコンにしてツイートをタイムラインに表示するもの。
評価極性辞書がユーザごとに作られるため、その人の特性にあったツイート評価がされる。
